1. Go to https://dev.azure.com/alexymarketplace
2. User settings â†’ Personal Access Tokens â†’ New Token
3. Scope: **Marketplace (Publish)** â†’ **Read, create, update, and delete extensions**

### 5. Publish the VSIX

Or publish directly (rebuilds and publishes):
```powershell

```
